# Steamdeck-Korean-input
A simple QWERTY-to-Hangul converter for Steam Deck Desktop Mode. Instantly transforms English keystrokes into Korean (e.g., "dkssud" → "안녕"). Perfect for Bluetooth keyboard users seeking seamless typing on SteamOS without IME hassle.

사용법
1. zip 압축 풀기
2. Konsole에서 cd KoreanInput
3. ./install.sh 실행
4. 프로그램 열고 "한영키 등록" 버튼 클릭 → 한영키 누르면 자동 등록
